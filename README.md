# Chalk & Co ✨

A modern frontend project built with React.

## 🌐 Live Demo

https://chalk-n-co.vercel.app

---

## About This Project

I built Chalk & Co while learning frontend development and exploring modern React workflows.

The main idea behind this project was to experiment, build fast, break things, fix them, and understand how real frontend projects come together.

For this project, I also tried **vibe coding with Antigravity** to speed up the UI building process. Instead of just generating everything and leaving it as it is, I kept modifying components, fixing issues, changing layouts, and improving parts of the project myself while learning along the way.

This project helped me understand:
- React component structure
- Responsive layouts
- Frontend project organization
- Debugging and fixing UI issues
- How modern frontend workflows actually feel in practice

## 📦 Running Locally

Clone the repository:

```bash
git clone <your-repo-url>
```

Go to the project folder:

```bash
cd chalk-n-co
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```

---

## 🔧 Production Build

```bash
npm run build
```
